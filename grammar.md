# OOPS Grammar

## Class-level syntax

```text
class filename(<A, B, ...>)? {
    (private|protected|public) type variable;

    (private|protected|public) filename(<type1, type2, ...>)?(type1 parameter1, type2 parameter2, ...) {
        ...
    }

    (private|protected|public) def method_name(<type1, type2, ...>)?(type1 parameter1, type2 parameter2, ...) -> return_type {
        ...
    }
}
```
